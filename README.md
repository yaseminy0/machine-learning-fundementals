# PCA and Generative Modeling

This section includes projects focused on dimensionality reduction, visualization, and generative modeling.

## Projects

### 1. PCA and t-SNE on Fashion-MNIST
This project applies PCA and t-SNE to the Fashion-MNIST dataset for dimensionality reduction and 2D visualization. It also explores image generation through PCA-based reconstruction.

### 2. Face Generation with GMM and PCA
This project combines PCA and Gaussian Mixture Models (GMM) to generate synthetic face images. PCA is used to reduce dimensionality before fitting GMM, making the generative process more efficient.

## Techniques Used
- PCA
- t-SNE
- Gaussian Mixture Models
- Image reconstruction
- Data visualization
